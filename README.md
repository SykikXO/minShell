minShell
a quickshell config for minimal functional sidebar on hyprland.
popup window for wifi and bluetooth setup.

how to use:

create a quickshell directory/folder in .config
clone using -> git clone https://www.github.com/sykikxo/minshell.git
go into dir/folder -> cd minShell
move all files to quickshell dir -> mv * to ~/.config/quickshell/
start quickshell from term, or add to hyprland.conf
